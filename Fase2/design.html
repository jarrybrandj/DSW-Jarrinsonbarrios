El proposito de este sitio es fomentar los aprendizajes de hojas de estilos en cascadas CSS, Frameworks CSS, gracias al curso de diseño web, en la fase 2 diseño y construccion.

Hojas de estilo en cascada CSS

Las hojas de estilo en cascada (CSS, Cascading Style Sheets = Hojas de estilo) que contempla definiciones de formato (tipo de fuente, tamaño, color de la fuente, color de fondo, párrafos, etc) de las distintas etiquetas que forman una página *.HTML. En otas palabras Las hojas de estilo en cascada (CSS) ofrecen un modo flexible de agregar estilos a las páginas de su sitio Web. Este conjunto de reglas de formato rige el aspecto de sus páginas y le permite definir las fuentes, los colores, el formato y otras características de presentación.

Frameworks CSS

Un framework de CSS es una biblioteca de estilos genéricos que puede ser usada para implementar diseños web. Aportan una serie de utilidades que pueden ser aprovechadas frecuentemente en los distintos diseños web. Tambien Podemos decir que son un conjunto de componentes para interfaz de usuario que te permite desarrollar aplicaciones basadas en HTML de manera rápida y efectiva. Todos los componentes son compatibles en móviles y diversos navegadores web.
